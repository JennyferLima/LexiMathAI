# LexiMath IA

Protótipo funcional desenvolvido para a disciplina de Informática na Educação.

## Objetivo

O sistema simula um fluxo adaptativo para apoio à interpretação de problemas matemáticos verbais nos anos iniciais.

O foco da proposta é diferenciar:

- erros interpretativos;
- dificuldades de compreensão textual;
- erros matemáticos procedimentais.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript puro

## Funcionalidades

- fluxo do aluno;
- análise simulada;
- intervenção adaptativa;
- dashboard do professor;
- revisão docente;
- taxonomia de erros.

## Execução local

Basta abrir o arquivo `index.html` no navegador.

## Hospedagem no GitHub Pages

1. Criar repositório no GitHub.
2. Enviar os arquivos.
3. Ir em:

Settings → Pages

4. Selecionar:
- Branch: main
- Pasta: root

5. Salvar.

O GitHub Pages irá gerar um link público automaticamente.

## Observação metodológica

Este protótipo representa uma simulação conceitual baseada em regras simples e dados mockados.

Não há modelo real de NLP treinado nesta etapa.

A avaliação atual é qualitativa e exploratória.

## Fluxo de navegação do protótipo

O sistema foi desenvolvido como um protótipo navegável de demonstração do fluxo metodológico proposto.

### Módulo do aluno

1. Na tela inicial, clique em:
   - `Entrar como aluno`

2. O sistema redirecionará para:
   - `Novo Desafio`

3. Após responder o problema, clique em:
   - `Analisar minha resposta`

4. O sistema exibirá:
   - diagnóstico interpretativo;
   - evidências utilizadas;
   - elementos não considerados;
   - sugestão adaptativa simulada.

---

### Módulo do professor

1. Retorne à tela inicial do sistema.
2. Clique em:
   - `Entrar como professor`

3. O sistema exibirá:
   - métricas simuladas da turma;
   - categorias de dificuldade;
   - padrões recorrentes;
   - sugestões pedagógicas.

---

### Observação

O protótipo prioriza a demonstração do fluxo conceitual, da experiência de interação e da proposta metodológica da plataforma.
